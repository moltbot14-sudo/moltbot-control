# MoltBot Control

Sistema para conectar dos Ubuntu con OpenClaw, permisos elevados, control cruzado, panel móvil, automejora 24/7 e integraciones Drive/Gmail/GitHub.

## Objetivo

- PC 1: Enrique-PC
- PC 2: Miki-PC
- Ambos con OpenClaw.
- Ambos con sudo sin contraseña.
- Ambos accesibles por Tailscale.
- Ambos controlables por SSH.
- App móvil PWA para ver estado, progreso, logs y tareas.
- Automejora 24/7 con backlog, logs y búsqueda de skills.

## Estados del panel móvil

- Verde: terminado/libre.
- Azul: trabajando.
- Rojo: error.
- Negro: apagado/no responde.
